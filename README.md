# Time Tracking

Time Tracking is a lightweight task and project management app that helps users create projects and tasks, then log time entries to track how much time is spent on each task. It provides a clear history of time logs to support better time management and productivity.

## Key Features
- Create and manage projects
- Create and manage tasks
- Log time entries per task
- View a clear history of time logs
- Local storage support
